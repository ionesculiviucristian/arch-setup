Neovim Cheatsheet — Leader key is Space

Quick reference for this Neovim configuration. Press Space and wait to see all available keybindings via which-key.

Tip: Press Space ? to show all buffer-local keymaps. Start typing any Space combo and which-key will show you what's available.

Basics

i Vim Modes

KeyAction
iEnter Insert mode (type text)
EscReturn to Normal mode
vEnter Visual mode (select text)
VVisual Line mode (select lines)
:Enter Command mode

W Files & Buffers

KeyAction
:wSave current file
:qQuit (close window)
:wqSave and quit
:q!Quit without saving
Space nNext buffer
Space pPrevious buffer
Space xClose current buffer

M Movement

KeyAction
h j k lLeft, Down, Up, Right
w / bNext / previous word
0 / $Start / end of line
gg / GTop / bottom of file
{ / }Previous / next paragraph
Ctrl+u / dHalf page up / down

E Editing

KeyAction
ddDelete (cut) line
yyYank (copy) line
pPaste after cursor
uUndo
Ctrl+rRedo
Space yYank to system clipboard
gcToggle comment (select + gc, or gcc for line)

/ Search

KeyAction
/textSearch forward for "text"
?textSearch backward for "text"
n / NNext / previous match
*Search word under cursor

S Surround (nvim-surround)

KeyAction
ysmotioncharAdd surround (e.g. ysiw" wraps word in quotes)
dscharDelete surround (e.g. ds")
csoldnewChange surround (e.g. cs"')

File Explorer & Finder (snacks.nvim)

D Explorer

KeyAction
Space eToggle file explorer sidebar
Space FReveal current file in explorer

P Picker (Fuzzy Finder)

KeyAction
Space f fFind files by name
Space f gGrep (search text in all files)
Space f bList open buffers
Space f hSearch help pages

Terminal

> Floating Terminal

KeyAction
Ctrl+\Toggle floating terminal (works in normal & terminal mode)

Code Intelligence (LSP)

L LSP Navigation

KeyAction
KHover info (type, docs) for symbol under cursor
gdGo to definition
gDGo to declaration
grList references
giGo to implementation
]] / [[Jump to next / previous word reference
Note: These are Neovim's built-in LSP keybindings (0.11+). They work automatically when a language server is attached.

A LSP Actions

KeyAction
grnRename symbol
graCode action (quick fix, refactor)
grrList references
Format on save is enabled automatically via conform.nvim (Prettier, Black, etc.).

C Autocompletion (blink.cmp)

KeyAction
TabSelect next suggestion / jump in snippet
Shift+TabSelect previous suggestion / jump back in snippet
EnterAccept selected suggestion
Ctrl+SpaceTrigger completion manually
Ctrl+eDismiss completion menu
Note: Completions appear automatically as you type. Cmdline completion works in : command mode too.

Diagnostics & Trouble

! Diagnostics

KeyAction
Space x xToggle all diagnostics panel
Space x XToggle current buffer diagnostics only
Space c sToggle symbols outline
Space c lToggle LSP definitions/references panel
Space x LToggle location list
Space x QToggle quickfix list
Tip: Use :lua vim.diagnostic.open_float() to see the full error message for the current line.

Code Folding (nvim-ufo)

F Folds

KeyAction
zaToggle fold under cursor
zROpen all folds
zMClose all folds
zo / zcOpen / close single fold

Git

G Git Integration

KeyAction
:DiffviewOpenOpen diff view for all changes
:DiffviewCloseClose diff view
:DiffviewFileHistory %File history for current file
Gitsigns shows added/modified/deleted lines in the gutter automatically.
